## 3. Требования

### 3.1 Функциональные требования

1. **Разделение потока** на две симметричные ветви с соотношением 50/50
2. **Минимизация гидравлических потерь** (целевой КПД ≥ 90%)
3. **Выпрямление закрученного потока** от импеллера
4. **Равномерное распределение** скорости по сечению на выходах
5. **Отсутствие кавитации** при рабочих параметрах

### 3.2 Геометрические ограничения

| Параметр | Требование |
|----------|------------|
| Диаметр входа | 85 мм (фиксирован) |
| Диаметр выходов | 44–61 мм (под сопла) |
| Угол разделения | ≤ 25° (оптимально 15–20°) |
| Общая длина | По возможности компактно |

### 3.3 Производственные требования

- Изготовление методом 3D-печати (FDM/SLA)
- Материал: PETG, ASA или смола
- Минимальная толщина стенки: 2 мм
- Возможность печати без поддержек (желательно)

## 4. Анализ проблем исходной конструкции

### 4.1 Выявленные недостатки

| Проблема | Влияние | Критичность |
|----------|---------|-------------|
| R_round = 0 (скругления отключены) | Резкие углы, отрыв потока | Высокая |
| Малый радиус поворота (10 мм) | R/D = 0.16, норма ≥ 1.5 | Высокая |
| Угол разделения 30° | Повышенные потери | Средняя |
| Резкое сужение на выходе (угол ~27°) | Турбулизация | Средняя |
| Отсутствие спрямляющего аппарата | Неравномерное распределение | Высокая |
| Острый разделительный клин (hull) | Отрыв, рециркуляция | Высокая |

### 4.2 Влияние закрутки потока

Без выпрямления закрученный поток от импеллера:
- Неравномерно распределяется между ветвями (асимметрия 10–20%)
- Создаёт вторичные течения в поворотах
- Увеличивает потери давления на 15–25%

## 5. Предлагаемое решение

### 5.1 Общая концепция

```
                    ┌─────────────────────────────────────────┐
                    │                                         │
  Импеллер    Входной     Статор +        Зона           Выходные
     →        участок    Сплиттер     разделения         ветви
                    │                                         │
              ══════╪═══════╪════════════╪═══════════════╪════
                    │       │            │               │
                  50мм    80мм        ~100мм          200мм+
                    │                                         │
                    └─────────────────────────────────────────┘
```

### 5.2 Ключевые улучшения

#### 5.2.1 Спиральный статор с постепенным выпрямлением

**Конструкция:**
- 4 боковые лопатки + 1 центральная (переходящая в сплиттер)
- Входной угол лопаток: 14° (соответствует закрутке потока)
- Выходной угол: 0° (полное выпрямление)
- Закон изменения угла: θ(z) = θ_in × (1 - z/L)² (ease-out)
- Длина лопаток: 60–80 мм

**Преимущества спиральных лопаток vs прямых под углом:**
- Угол атаки на входной кромке ~0° (vs 14° у прямых)
- Потери давления ниже на 15–25%
- Отсутствие отрыва потока
- Минимальный риск кавитации

#### 5.2.2 Центральный сплиттер

**Конструкция:**
- Продолжение центральной лопатки статора
- Профиль: эллиптический или NACA 0012
- Передняя кромка: скруглённая (R = 1–2 мм)
- Задняя кромка: острая (сходящаяся)
- Постепенное расширение в плоскости разделения

**Преимущества:**
- Гарантированное деление 50/50
- Плавный переход от выпрямления к разделению
- Уменьшение зоны рециркуляции

#### 5.2.3 Плавная зона бифуркации

**Параметры:**
- Угол разделения: 20° (вместо 30°)
- S-образный переход с ease-in-out интерполяцией
- Радиус поворота: 100 мм (R/D ≈ 1.64)

#### 5.2.4 Оптимизированное сужение на выходе

**Параметры:**
- Полуугол конуса: ≤ 7° (вместо 27°)
- Длина конического перехода: ~60–100 мм
- Стабилизирующий прямой участок после сужения

## 8. Реализация в OpenSCAD

### 8.1 Структура модели

```
model.scad
├── module spiral_stator_vane()      — одна спиральная лопатка
├── module stator_assembly()          — сборка статора (5 лопаток)
├── module central_splitter_vane()    — центральный сплиттер
├── module transition_improved()      — зона бифуркации
├── module y_separator_channel()      — основной канал
└── module y_separator_complete()     — полная сборка
```

### 8.2 Ключевые модули

#### Спиральная лопатка статора

```openscad
module spiral_stator_vane(
    R_outer = 42,           // внешний радиус
    R_inner = 10,           // радиус втулки
    L = 80,                 // длина лопатки
    twist_in = 14,          // угол закрутки на входе
    thickness = 2           // толщина лопатки
) {
    steps = 40;
    
    for (i = [0:steps-1]) {
        t = i / steps;
        t_next = (i + 1) / steps;
        
        z1 = t * L;
        z2 = t_next * L;
        
        // Ease-out: угол уменьшается от twist_in до 0
        angle1 = twist_in * pow(1 - t, 2);
        angle2 = twist_in * pow(1 - t_next, 2);
        
        // Толщина: утолщение в середине
        th1 = thickness * (0.5 + 0.5 * sin(t * 180));
        th2 = thickness * (0.5 + 0.5 * sin(t_next * 180));
        
        hull() {
            translate([0, 0, z1])
            rotate([0, 0, angle1])
            translate([R_inner, 0, 0])
            cube([R_outer - R_inner, max(th1, 0.5), 0.1]);
            
            translate([0, 0, z2])
            rotate([0, 0, angle2])
            translate([R_inner, 0, 0])
            cube([R_outer - R_inner, max(th2, 0.5), 0.1]);
        }
    }
}
```

#### Сборка статора

```openscad
module stator_assembly(
    n_side_vanes = 4,       // боковые лопатки
    R_outer = 42,
    R_inner = 8,
    L = 80,
    twist = 14
) {
    // Боковые лопатки (равномерно по окружности, смещены от центральной)
    for (i = [0:n_side_vanes-1]) {
        rotate([0, 0, i * 360/n_side_vanes + 360/(2*n_side_vanes)])
        spiral_stator_vane(
            R_outer = R_outer,
            R_inner = R_inner,
            L = L,
            twist_in = twist
        );
    }
    
    // Центральная лопатка (переходит в сплиттер)
    // Реализуется отдельно в central_splitter_vane
}
```

#### Центральный сплиттер

```openscad
module central_splitter_vane(
    Din = 85,
    L_stator = 80,          // длина части статора
    L_splitter = 60,        // длина части сплиттера
    twist_in = 14,
    thickness_max = 4,
    expansion_ratio = 1.5   // расширение сплиттера
) {
    L_total = L_stator + L_splitter;
    steps = 60;
    
    for (i = [0:steps-1]) {
        t = i / steps;
        t_next = (i + 1) / steps;
        
        z1 = t * L_total;
        z2 = t_next * L_total;
        
        // Угол закрутки (только в зоне статора)
        t_stator = min(t * L_total / L_stator, 1);
        t_stator_next = min(t_next * L_total / L_stator, 1);
        twist1 = twist_in * pow(1 - t_stator, 2);
        twist2 = twist_in * pow(1 - t_stator_next, 2);
        
        // Толщина (максимум в середине статора, сужение к концу)
        th1 = thickness_max * sin(min(t * 2, 1) * 180) * 
              (t < 0.5 ? 1 : (1 - (t - 0.5) * 0.6));
        th2 = thickness_max * sin(min(t_next * 2, 1) * 180) * 
              (t_next < 0.5 ? 1 : (1 - (t_next - 0.5) * 0.6));
        
        // Ширина (расширяется в зоне сплиттера)
        w1 = Din * 0.35 * (1 + max(0, (t - L_stator/L_total)) * 
             (expansion_ratio - 1) / (1 - L_stator/L_total));
        w2 = Din * 0.35 * (1 + max(0, (t_next - L_stator/L_total)) * 
             (expansion_ratio - 1) / (1 - L_stator/L_total));
        
        hull() {
            translate([0, 0, z1])
            rotate([0, 0, twist1])
            scale([w1, max(th1, 0.5), 1])
            cylinder(h = 0.1, r = 1, $fn = 16);
            
            translate([0, 0, z2])
            rotate([0, 0, twist2])
            scale([w2, max(th2, 0.5), 1])
            cylinder(h = 0.1, r = 1, $fn = 16);
        }
    }
}
```

## 10. Приложения

### А. Формулы для расчётов

**Число Рейнольдса:**
$$Re = \frac{V \cdot D}{\nu}$$

**Угол закрутки от импеллера:**
$$\theta = \arctan\left(\frac{V_t}{V_a}\right) = \arctan\left((1-slip) \cdot \tan(\beta_{blade})\right)$$

**Потери давления:**
$$\Delta P = \zeta \cdot \frac{\rho V^2}{2}$$

**Коэффициент потерь Y-разделителя:**
$$\zeta = \zeta_{вход} + \zeta_{статор} + \zeta_{разделение} + \zeta_{поворот} + \zeta_{сужение}$$

### Б. Справочные данные

- Плотность воды при 20°C: ρ = 998 кг/м³
- Кинематическая вязкость воды: ν = 1.004 × 10⁻⁶ м²/с
- Давление насыщенных паров (20°C): 2.34 кПа
