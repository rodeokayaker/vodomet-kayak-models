## Модели для водомёта каяка

Этот репозиторий содержит параметрические 3D‑модели (OpenSCAD) узлов водомёта для каяка:

- разделители и выпрямители потока;
- сопла и экспериментальные варианты геометрии;
- вспомогательные модули и сборки.

### Содержимое

- `separator/straightener140.scad` — модуль выпрямителя/остановки вращения потока в вертикальной трубе.
- `separator/model.scad` — параметрическая модель внутреннего канала Y‑разделителя с адаптерными кольцами.
- `separator/adapter_plate_common.scad` и `separator/common_params.scad` — общие модули адаптерного кольца и базовых размеров (диаметры труб, ушей, паза под резинку и т.п.), которые переиспользуются в стрейтенере и разделителе.
- `soplo_1.scad` — модель сопла (геометрия для экспериментов с тягой и КПД).
- `TZ_Soplo.md`, `reshenie_1.md` — текстовые описания требований и вариантов решения.

### Использование

Модели создаются в OpenSCAD и рассчитаны под 3D‑печать.

Рекомендуемый рабочий процесс:

1. Открыть интересующий `.scad`‑файл в OpenSCAD.
2. При необходимости скорректировать параметры (диаметры, высоты, число лопастей и т.п.).
3. Сгенерировать STL и передать в слайсер.
